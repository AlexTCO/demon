# Vampire the Masquerade V20 System

[![Foundry Shield 0.8.8]][Foundry URL]
[![MIT License]][MIT URL]
[![GitHub Super-Linter]][Super-Linter URL]

This system is for playing Demon the fallen on the Foundry Virtual Tabletop System. (Alfa version, not playable yet)

The system is based on Vampire the Masquerade V20 System (https://github.com/Layman2501/wod20)

Licensed materials will NOT be provided (list of powers, Houses, etc.) as this is a system provided for public use for free.
